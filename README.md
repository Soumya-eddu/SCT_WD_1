# Responsive Landing Page 🌐

A modern and responsive landing page developed using HTML, CSS, and JavaScript with interactive sections and smooth user experience.

## Features

* Responsive Navigation Bar
* Scroll-Based Navbar Color Change
* Mobile Menu Toggle
* Smooth Scrolling
* Interactive Buttons
* Animated Counters
* Project Filtering
* FAQ Section
* Responsive Design
* Modern UI with Hover Effects

## Technologies Used

* HTML
* CSS
* JavaScript
* Font Awesome

## Project Files

* `index.html` – Webpage structure
* `style.css` – Styling and responsive design
* `script.js` – Interactive functionality
